# iPhone install guide — v3.1 hotfix

Upload/replace these files in GitHub:

- app.py
- goat_engine_live.py
- requirements.txt
- README.md
- README_iPhone_Deploy.md

Then:
1. Commit changes.
2. Streamlit > Manage app > Reboot app.
3. Open app.
4. You should see: GOAT Shield Live v3.1
