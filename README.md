# GOAT Shield Live v3 — Best Price Board

Phone-friendly live odds scanner for paper-betting research only.

## v3 upgrades

- Best Price Board across bookmakers returned by The Odds API.
- Compares the same pick/market/line across bookmakers.
- Shows best odds, best bookmaker, Pinnacle odds if present, TAB/Betcha odds if present in the feed, Bet365 odds if present, average odds, edge, and closest misses.
- Supports h2h/moneyline, spreads, and totals.
- Can scan selected active sports.
- Can filter to today in NZ time or next 24 hours.
- Keeps paper-only safety.

## What this app does NOT do

- No real-money betting.
- No auto-login to TAB / Bet365 / Pinnacle.
- No sportsbook scraping.
- No bank/card details.
- No bypassing sportsbook rules.
